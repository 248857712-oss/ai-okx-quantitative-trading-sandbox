## 该文件旨在给模型建立参数调节


```
├── 币安4000条测试集+验证集     会读取近4000条(6个月左右)1hk线
├── 模型调参数.py   用于自动给出合适的模型参数 
├── 策略回测.py     用于回测模型参数,给出结果  
  
 
```
